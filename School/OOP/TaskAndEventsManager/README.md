# Система за управление на лични задачи и събития
## Personal Task & Event Manager

### Описание
Конзолно приложение за управление на задачи, събития, домашни работи и изпити. Приложението използва основни концепции на обектно-ориентираното програмиране (ООП) като наследяване, енкапсулация и полиморфизъм.

### Структура на проекта

Проектът съдържа следните класове:

1. **Priority.java** - Enum за нивата на приоритет (LOW, MEDIUM, HIGH)
2. **CalendarItem.java** - Абстрактен базов клас за всички типове активности
3. **Task.java** - Клас за задачи
4. **Event.java** - Клас за събития
5. **Homework.java** - Клас за домашни работи
6. **Exam.java** - Клас за изпити
7. **Calendar.java** - Клас за управление на календара
8. **TaskManagerApp.java** - Главен клас с потребителски интерфейс

### Компилиране

За да компилирате проекта, отворете терминал/команден прозорец в директорията с файловете и изпълнете:

```bash
javac *.java
```

### Стартиране

За да стартирате приложението:

```bash
java TaskManagerApp
```

### Функционалности

#### 1. Добавяне на активност
- Изберете тип активност (Задача, Събитие, Домашно, Изпит)
- Въведете общи данни: заглавие, описание, дата, час, локация, приоритет
- Въведете специфични данни според типа активност

#### 2. Премахване на активност
- Въведете заглавието на активността, която искате да премахнете

#### 3. Търсене на активности
- По дата - покажи всички активности на определена дата
- По тип - филтрирай по тип активност (ЗАДАЧА/СЪБИТИЕ/ДОМАШНО/ИЗПИТ)
- По приоритет - филтрирай по ниво на приоритет

#### 4. Преглед на предстоящи активности
- Показва всички незавършени активности от днес нататък
- Сортирани по дата и час

#### 5. Преглед на завършени активности
- Показва всички маркирани като завършени активности

#### 6. Маркиране като завършена
- Маркира избрана активност като завършена

#### 7. Преглед на календара
- Показва всички активности, сортирани по дата

### Формати на данни

- **Дата**: дд.мм.гггг (например: 15.01.2026)
- **Час**: чч:мм (например: 14:30)
- **Приоритет**: 1 (Нисък), 2 (Среден), 3 (Висок)

### Специфични атрибути по тип активност

#### Задача (Task)
- Очаквана продължителност (минути)
- Категория (напр. "Работа", "Лично")

#### Събитие (Event)
- Краен час
- Списък с участници

#### Домашно (Homework)
- Учебен предмет
- Краен срок
- Зададено от
- Проверка за просрочване

#### Изпит (Exam)
- Учебен предмет
- Продължителност (минути)
- Зала/стая
- Възможност за добавяне на оценка

### Обработка на грешки

Приложението валидира:
- Формат на дата и час
- Валидност на избора от менюта
- Оценки за изпити (2.00-6.00)
- Съществуване на активности при операции
