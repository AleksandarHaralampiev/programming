# Задание на проект

## 1. Задание

**Система за управление на лични задачи и събития (Personal Task & Event Manager)**

## 2. Описание на задачата

Приложение за управление на лични задачи и събития, което да позволява на потребителите да организират ежедневието си чрез добавяне на различни типове активности (задачи, събития, домашни, изпитвания) към календар.
## 3. Изисквания

### Клас "CalendarItem" (абстрактен базов клас):

**Атрибути:**

- `title`: Заглавие на активността (String)
- `description`: Описание на активността (String)
- `date`: Дата на активността (LocalDate)
- `time`: Час на активността (LocalTime)
- `location`: Място на провеждане (String)
- `priority`: Приоритет (Enum: LOW, MEDIUM, HIGH)
- `isCompleted`: Статус на завършеност (boolean)

**Методи:**

- Конструктор за инициализация на обектите
- Гетери и сетери за всички атрибути
- `abstract String getType()`: Абстрактен метод за връщане типа на активността
- `markAsCompleted()`: Метод за маркиране като завършена
- `toString()`: Метод за форматиран изход

### Клас "Task" (наследява CalendarItem):

**Допълнителни атрибути:**

- `estimatedDuration`: Очаквана продължителност в минути (int)
- `category`: Категория на задачата (String) - напр. "Работа", "Лично"

**Методи:**

- Конструктор
- `getType()`: Връща "Задача"

### Клас "Event" (наследява CalendarItem):

**Допълнителни атрибути:**

- `endTime`: Крайно време на събитието (LocalTime)
- `participants`: Списък с участници (ArrayList `<String>`)

**Методи:**

- Конструктор
- `addParticipant(String participant)`: Добавя участник
- `getType()`: Връща "Събитие"

### Клас "Homework" (наследява CalendarItem):

**Допълнителни атрибути:**

- `subject`: Учебен предмет (String)
- `dueDate`: Краен срок (LocalDate)
- `assignedBy`: От кого е зададено (String)

**Методи:**

- Конструктор
- `isOverdue()`: Проверява дали е просрочено
- `getType()`: Връща "Домашно"

### Клас "Exam" (наследява CalendarItem):

**Допълнителни атрибути:**

- `subject`: Учебен предмет (String)
- `duration`: Продължителност в минути (int)
- `room`: Зала/стая (String)
- `grade`: Оценка (Double) - nullable

**Методи:**

- Конструктор
- `setGrade(Double grade)`: Задава оценка
- `getType()`: Връща "Изпит"

### Клас "Calendar":

**Атрибути:**

- `items`: Списък от календарни активности (ArrayList `<CalendarItem>`)

**Методи:**

- `addItem(CalendarItem item)`: Добавя нова активност
- `removeItem(String title)`: Премахва активност по заглавие
- `findItemsByDate(LocalDate date)`: Търси активности по дата
- `findItemsByType(String type)`: Филтрира по тип активност
- `listUpcomingItems()`: Показва предстоящи активности
- `listCompletedItems()`: Показва завършени активности
- `sortByDate()`: Сортира активностите по дата
- `listItemsByPriority(Priority priority)`: Филтрира по приоритет

### Enum "Priority":

- `LOW`: Нисък приоритет
- `MEDIUM`: Среден приоритет
- `HIGH`: Висок приоритет

### Основен клас "TaskManagerApp":

Съдържа `main()` метод и логика за управление на потребителския интерфейс.

## 4. Описание на интерфейс към потребителите

Създайте конзолен потребителски интерфейс с главно меню, което предлага следните опции:

**Главно меню:**

1. Добави нова активност
   - Избор на тип (Задача / Събитие / Домашно / Изпит)
   - Въвеждане на съответните данни за избрания тип
2. Премахни активност (по заглавие)
3. Търси активности
   - По дата
   - По тип
   - По приоритет
4. Покажи всички предстоящи активности
5. Покажи завършени активности
6. Маркирай активност като завършена
7. Преглед на календара (сортиран по дата)
8. Изход от програмата

**Примерен изход при преглед:**

```
=== ПРЕДСТОЯЩИ АКТИВНОСТИ ===
[ИЗПИТ] Математика | 15.01.2026 10:00 | Зала 205 | Приоритет: HIGH
[ДОМАШНО] Есе по литература | Краен срок: 18.01.2026 | Приоритет: MEDIUM
[СЪБИТИЕ] Среща с приятели | 20.01.2026 18:30 | Кафене Облак
```
